## Fuer den schnellen Einstieg

1. Release herunterladen: [Latest Release](https://github.com/breiti35/ets2-overlay-feed/releases/latest)
2. App starten
3. In `System -> Update & Diagnose` den Update-Status pruefen
4. In `Twitch Setup` oder `YouTube Setup` Bot konfigurieren

Die App speichert Einstellungen lokal und laedt sie beim naechsten Start automatisch.
