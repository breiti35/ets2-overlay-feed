## Für den schnellen Einstieg

1. Release herunterladen: [Latest Release](https://github.com/breiti35/ets2-overlay-feed/releases/latest)
2. App starten
3. In `System -> Update & Diagnose` den Update-Status prüfen
4. In `Twitch Setup` oder `YouTube Setup` Bot konfigurieren

Die App speichert Einstellungen lokal und lädt sie beim nächsten Start automatisch.
